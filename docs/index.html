Modules

activerdf Lua ActiveRDF is a library for accessing RDF data from Lua programs.
activerdf.ActiveRdfAdapter Generic superclass of all adapters
activerdf.ConnectionPool Maintains pool of adapter instances that are connected to datasources.
activerdf.FederationManager Manages the federation of datasources: distributes queries to right datasources and merges their results.
activerdf.Literal Represents an RDF literal, optionally datatyped.
activerdf.NTriplesParser Ntriples parser
activerdf.Namespace Manages namespace abbreviations and expansions.
activerdf.ObjectManager Constructs LOOP classes for RDFS classes (in the right namespace).
activerdf.Query Represents a query on a datasource, abstract representation of SPARQL features.
activerdf.Query2Jars2 Translates abstract query into jars2 query.
activerdf.Query2SPARQL Translates abstract query into SPARQL that can be executed on SPARQL-compliant data source.
activerdf.RDFS.Resource Represents an RDF resource.
activerdf.string Extends the default lua string functions.
activerdf.table Extends the default lua table functions.
activerdf_rdflite.RDFLite Lua RDFLite is a small RDF datastore, based on sqlite3.
activerdf_sparql.SparqlAdapter SPARQL adapter

SparqlAdapter is a class (simulated by LOOP).
