all (tbl, func)
passes each element of the table to the given function and returns true if the function never returns false or nil. If the function is not given, it adds an implicit function(i,obj) return obj end. That is, table.all will return true only if none of the tbl members are false or nil.

any (tbl, func)
passes each element of the table to the given function and returns true if the function ever returns a value other that false or nil. If the function is not given, it adds an implicit function(i,v) return v end. That is, table.any will return true if at least one of the tbl members is not false or nil.

inject (init, tbl, func)
combines the elements of tbl by applying the function func to an accumulator value (memo) and each element in turn. At each step, memo is set to the value returned by the function func. You can supply an initial value for memo using the parameter init. If init is ommited it uses the first element of the table as a the initial value (and skips that element while iterating).

Parameters

  • init: initial value for memo.
  • tbl: a table.
  • func: function( memo, element ) ... end

Usage:

-- Sum some numbers
table.inject ( {1,2,3}, function(sum, n) return sum + n end ) --> 6

-- Multiply some numbers
table.inject (1, {1,2,3,4}, function(product, n) return product * n end ) --> 24

-- Multiple tables intersection
t = {{1,2,3}, {2,3}, {3,4}}
table.inject( t, function( intersec, n ) return table.intersection(intersec, n) end ) --> { 3 }
