Module activerdf.RDFS.Resource

Represents an RDF resource. Manages manipulations of that resource, including data lookup (e.g. john.age), data updates (e.g. john.age=20), class-level lookup (Person:find_by_name 'john'), and class-membership (oo.classof(john) ...Person).

Dynamic attribute-based finders

Dynamic attribute-based finders work by appending the name of an attribute to find_by_, so you get finders like Person:find_by_name(name), Person:find_by_last_name(last_name), Payment:find_by_transaction_id(transaction_id).
So instead of writing Person:find( { where = { name = "john" } } ), you just do Person:find_by_name("john"). It‘s also possible to use multiple attributes in the same find by separating them with "and", so you get finders like Person:find_by_user_name_and_password(user_name, password) or even Payment:find_by_purchaser_and_state_and_country(purchaser, state, country).
