Module activerdf.Namespace

Manages namespace abbreviations and expansions.

Release: $Id$

Functions

abbreviations () returns currently registered namespace abbreviations (e.g 'foaf', 'rdf').
expand (prefix, localname) returns URI (string) formed by concatenation of prefix and localname.
localname (resource) returns local-part of URI.
lookup (prefix, localname) returns a resource whose URI is formed by concatenation of prefix and localname.
prefix (resource) returns prefix (if known) for the non-local part of the URI, or nil if prefix not registered.
register (prefix, fullURI) registers a namespace prefix and its associated expansion (full URI).
