Module activerdf.string

Extends the default lua string functions.

Release: $Id$

Functions

capitalize (str) returns a copy of str with the first character converted to uppercase and the remainder to lowercase.
chop (str) returns a new string with the last character removed.
split (str, pattern) divides str into substrings based on a delimiter (pattern), returning an table of these substrings.

chop (str)
returns a new string with the last character removed. If the string ends with \r\n, both characters are removed. Applying chop to an empty string returns an empty string. string.chomp is often a safer alternative, as it leaves the string unchanged if it doesn’t end in a record separator.

Parameters

  • str:
split (str, pattern)
divides str into substrings based on a delimiter (pattern), returning an table of these substrings.

Parameters

  • str: input string.
  • pattern: pattern delimiter.

Usage

  • string.split(" now's the time", " ") --> { "now's", "the", "time" }
  • string.split("1, 2.34,56, 7", "[,%s]+") --> { "1", "2.34", "56", "7" }
  • string.split("mellow yellow", "ello" --> { "m", "w y", "w" }
  • string.split("1,2,,3,4,,", ",") --> { "1", "2", "", "3", "4" }
